## Arbitrary Precision Calculator Documentation
1. Introduction
An Arbitrary Precision Calculator (APC) is a calculator that can perform arithmetic operations on very large numbers that cannot be stored using normal C data types such as int or long long. In this project, each digit of the number is stored in a Doubly Linked List, allowing calculations on numbers of any length.

## 2. Objective
To perform arithmetic operations on very large numbers.

To overcome the limitations of built-in data types.

To implement linked list concepts in a real-world application.

## 3. Working
The user enters two large numbers and selects an arithmetic operation such as addition, subtraction, multiplication, or division. Each digit of the numbers is stored in a Doubly Linked List. The selected operation is performed digit by digit, and the final result is displayed.

## 4. Data Structure Used
This project uses a Doubly Linked List.

Each node stores:

One digit

Address of previous node

Address of next node

This makes it easy to traverse the number from both left and right while performing arithmetic operations.

## 5. Features
Supports numbers of any length.

Performs Addition, Subtraction, Multiplication, and Division.

Uses dynamic memory allocation.

Prevents overflow problems.

## 6. Applications
Scientific calculations

Banking software

Cryptography

Financial applications

Large integer computations

## 7. Advantages
No limitation on the number of digits.

Efficient storage using linked lists.

Easy to extend with more mathematical operations.

Provides better understanding of data structures.

## 8. Conclusion
The Arbitrary Precision Calculator successfully performs arithmetic operations on very large integers that cannot be handled by normal C data types. By using a Doubly Linked List, the project stores each digit separately and processes numbers of virtually unlimited length. This project demonstrates the practical use of linked lists, dynamic memory allocation, and arithmetic algorithms in solving real-world problems.
